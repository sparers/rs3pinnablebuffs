Setup

  • Make sure your interface scalling is 100% and your buff/debuff bar is visible and situated on a dark background (eg notes interface).
  • For the target debuffs tracker to work properly, pin the target info to a corner of the screen.
  • To set the buffs overlay position, use the overlay buttons in the settings tab.

    • How it works

    Using alt1 capture API, the plugin reads the information on your screen, in particular the buff/debuff bars and target info.

    Buffs with duration ≤59s update in real-time. Buffs >60s (e.g., 5 minutes) start counting down once the game updates their timer (e.g., when it changes from 5m to 4m).
